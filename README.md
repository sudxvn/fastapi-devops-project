# fastapi-devops-project

Pet-проект: Docker-контейнеризация FastAPI с PostgreSQL для портфолио DevOps
Этот проект — демонстрация навыков контейнеризации и оркестрации микросервисов. Я упаковал веб-приложение на FastAPI и базу данных PostgreSQL в отдельные контейнеры и настроил их взаимодействие.
Использовались такие базовые принципы как:контейнеризация, оркестрация, управление конфигурацией
Проект состоит из 2 сервисов:
API (веб приложение на FASTAPI) собирается из dockerfile
DB (база данных PostgreSQL) готовый официальный образ `postgres:16-alpine`
Контейнеры общаются друг с другом через внутреннюю сеть Docker Compose

# Использовал такой стек:
- **Backend:** Python 3.11, FastAPI, Uvicorn
- **База данных:** PostgreSQL 16
- **Контейнеризация:** Docker, Docker Compose
- **Среда:** WSL2 (Windows)

# Как запускается проект :
Заранее должен быть установлен Docker,Docker compose [Docker](https://www.docker.com/products/docker-desktop/)
Использумем командную строку (обязательно запуская от прав администратора)
Клонируйте репозиторий:
```bash
   git clone https://github.com/ТВОЙ_ЛОГИН/НАЗВАНИЕ_РЕПОЗИТОРИЯ.git
Папка проекта:
   cd НАЗВАНИЕ_ПАПКИ_ПРОЕКТА
Запуск сборки контейнера:
docker compose up -d --build
Откройте в браузере интерактивную документацию API:
http://localhost:8000/docs
